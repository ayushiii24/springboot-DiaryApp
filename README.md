# Secure Diary API

A backend REST API for a secure personal diary built using Spring Boot and Spring Security.
This project focuses on authentication, authorization, and user ownership of data.

# Features

- User Registration and Login
- Secure password hashing using BCrypt
- Authentication and authorization with Spring Security
- User-specific diary ownership (users can access only their own entries)
- CRUD operations for diary entries
- RESTful API design
- Clean backend architecture

# Tech Stack

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Maven

# Security Highlights

- Passwords are encrypted using BCrypt
- Protected endpoints using authentication
- Ownership-based authorization
- Secure API access
